# 프로젝트 마일스톤 목록

이 파일은 MoreThanLog 프로젝트의 주요 마일스톤을 추적하고 관리합니다. 완료된 마일스톤은 관리자 확인 후 제거됩니다.

## 🎯 현재 진행 중인 마일스톤

### [ ] 이미지 프록시 안정화 (P0)
- **목표**: S3 presigned URL 400 오류 해결 및 이미지 프록시 신뢰성 확보
- **현재 상태**: 일부 presigned URL에서 origin 400 응답 발생
- **다음 단계**:
  - `logs/image-proxy-errors.jsonl` 분석하여 쿼리 파라미터 문제 확인
  - `scripts/test-unpack-cases.js`로 문제 재현 및 해결
  - curl/HEAD 요청으로 200 응답 검증
- **예상 완료**: 1-2일

### [ ] SSG 페이지 크기 최적화 (P1)
- **목표**: 대용량 페이지의 page-data 크기 128KB 이하로 완화
- **현재 상태**: `the-forward-forward-algorithm-summary` 등에서 ~207KB 관찰됨
- **다음 단계**:
  - `optimizeRecordMap` 공격적인 최적화 적용
  - 무거운 블록 클라이언트 측 lazy-load 전환
  - `yarn build` 후 페이지 크기 검증
- **예상 완료**: 2-4일

## 🔄 진행 예정 마일스톤

### [ ] 운영 로그 시스템 마이그레이션 (P2)
- **목표**: JSONL 로그를 Sentry/중앙 로그 DB로 연동
- **현재 상태**: JSONL 파일로 기록 중
- **다음 단계**:
  - 민감 정보 마스킹 정책 확정
  - Sentry 또는 중앙 로그 DB 연동
  - 로그 집계 검증
- **예상 완료**: 2-3일

### [ ] CI/CD 자동화 강화 (P2)
- **목표**: GitHub Actions에 unwrap 테스트 자동화
- **현재 상태**: 로컬 스크립트 존재
- **다음 단계**:
  - GitHub Actions에 Node 테스트 job 추가
  - PR에서 회귀 테스트 실패 시 차단
- **예상 완료**: 1-2일

### [ ] 모니터링 및 알림 개선 (P3)
- **목표**: 이미지 프록시 실패 시 자동 알림 시스템
- **다음 단계**:
  - 실패 임계치 설정
  - Slack 알림 트리거 구현
- **예상 완료**: 1일

### [ ] 문서화 완성 (P3)
- **목표**: 운영 가이드 및 디버깅 문서 완성
- **다음 단계**:
  - image-proxy 동작 설명 문서 추가
  - 재현 가이드 및 체크리스트 작성
- **예상 완료**: 1-2일
